# Adding the YBT. Logo to Your Outlook Signature

You'll need the file `ybt-logo.html` — the YBT logo built in HTML with the brand font embedded.

## Step 1 — Open the logo in your browser

Double-click `ybt-logo.html`. It opens in your browser showing the YBT. logo on a dark block.

## Step 2 — Copy the logo

Click just before the logo, then select it (Cmd+A on Mac, Ctrl+A on Windows) and copy (Cmd+C / Ctrl+C).

## Step 3 — Open Outlook's signature editor

**Outlook for Mac (new Outlook):**
1. Outlook menu → Settings (or press Cmd+,)
2. Click **Signatures**
3. Click **+** to create a new signature, or double-click an existing one to edit it

**Outlook for Windows / Outlook on the web:**
1. Settings (gear icon) → **Accounts** → **Signatures**
2. Click **New signature**, or select an existing one

## Step 4 — Build the signature

1. Paste the logo (Cmd+V / Ctrl+V) into the signature editor
2. Add your details alongside or below it — name, title, mobile, email, and www.yourbusinesstoday.com
3. Give the signature a name (e.g. "YBT") and save

## Step 5 — Set it as your default

In the same signatures screen, choose the YBT signature under both **New messages** and **Replies/forwards**.

## Step 6 — Test it

Send yourself an email and check the logo appears correctly on the dark block with the red dot.

## One important note on the font

The logo uses Space Grotesk, the YBT brand typeface. For it to display in your Outlook editor, install the font on your machine first (free from Google Fonts — search "Space Grotesk", download, and double-click the font file to install). Recipients who don't have the font will see a clean fallback (Segoe UI or Arial) — the colours and layout stay identical either way.
